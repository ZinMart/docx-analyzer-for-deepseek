# АРХИТЕКТУРА ПРОЕКТА

## КОМПОНЕНТЫ
1. **main.py** - Главное окно программы (GUI)
2. **docx_analyzer.py** - Логика анализа DOCX файлов

## ПОТОК ДАННЫХ
1. Пользователь выбирает файл → main.py
2. main.py передает путь → docx_analyzer.py
3. Анализатор извлекает данные → возвращает результаты
4. GUI показывает статистику
